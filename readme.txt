== Description ==

Check pagerank of your blog pages automatically,display your real pagerank with an icon instantly.
The pagerank icon is running code from website checker site to generate. 
Visit http://www.websitechecker.info for details.

== Installation ==

1. Upload all flies to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the widget named "PageRank Icon" to sidebar, choose a icon style and save it.
